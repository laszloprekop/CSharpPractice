# CSharpPractice

Self-study and preparation work for the Lexicon .NET fullstack developer course teacher interview.

This public repository documents the learning journey through C# fundamentals and a fullstack project built independently before the course begins.

## Contents

### Challenges

Coding exercises to get familiar with C# syntax and problem-solving.

- **FizzBuzz** — classic FizzBuzz implemented as an interactive console app
- **TutorialTopLevelStatements** — exploration of C# top-level statements

### Kattis

Competitive programming solutions submitted to [open.kattis.com](https://open.kattis.com/).

- **FizzBuzz-Kattis** — FizzBuzz solved to Kattis specification and accepted online

### PolisIncidentMap — Fullstack Project

**Live demo: [polis-incidents-map.dentaku.se](https://polis-incidents-map.dentaku.se/)**

A fullstack web app displaying live Swedish police incidents on an interactive map. Adapted from an Android/Kotlin project built during the Linero course (module203).

- **Backend** — ASP.NET Core Web API (.NET 10) proxying the Polisen public API
- **Frontend** — React + Vite + Tailwind CSS with an interactive Leaflet map

See [`PolisIncidentMap/README.md`](PolisIncidentMap/README.md) for details.
