# PolisIncidentMap — Frontend

React + Vite + Tailwind CSS. Displays police incidents on an interactive map.

## Running locally

```bash
npm install
npm run dev
```

App will be available at `http://localhost:5173`.

## Build variables

| Variable | Description | Default |
|---|---|---|
| `VITE_API_URL` | Backend API base URL | `http://localhost:5116/api/incidents` |
